CatGPT API 中心
返回聊天

接口文档

POST /api/v1/messages 发送消息（支持 sessionId / deepThink / webSearch）

GET /api/v1/messages?sessionId=xxx 拉取消息列表与 pending 状态

鉴权：Authorization: Bearer cat_*** 或 X-API-Key: cat_***

API Key 使用教学

  1. 在本页创建 API Key（只会完整显示一次，请立即保存）。
  2. 调用 POST /api/v1/messages 发送消息，拿到 sessionId。
  3. 用 GET /api/v1/messages?sessionId=... 轮询获取回复。

建议轮询间隔：1~2 秒。

快速示例（curl）

# 1) 发送消息
curl -X POST "http://47.109.16.253:2333/api/v1/messages" \
  -H "Authorization: Bearer cat_xxx" \
  -H "Content-Type: application/json" \
  -d "{\"content\":\"你好\",\"deepThink\":true,\"webSearch\":false}"

# 返回中拿到 sessionId 后轮询
curl "http://47.109.16.253:2333/api/v1/messages?sessionId=你的sessionId" \
  -H "Authorization: Bearer cat_xxx"

Node.js 示例

const API_KEY = "cat_xxx";
const BASE = "http://47.109.16.253:2333";

async function run() {
  const send = await fetch(`${BASE}/api/v1/messages`, {
    method: "POST",
    headers: {
      "Authorization": `Bearer ${API_KEY}`,
      "Content-Type": "application/json"
    },
    body: JSON.stringify({
      content: "请给我一个健身计划",
      deepThink: true,
      webSearch: true
    })
  });
  const queued = await send.json();
  const sessionId = queued.sessionId;

  const timer = setInterval(async () => {
    const res = await fetch(`${BASE}/api/v1/messages?sessionId=${sessionId}`, {
      headers: { "Authorization": `Bearer ${API_KEY}` }
    });
    const data = await res.json();
    const last = data.messages[data.messages.length - 1];
    if (last && last.role === "assistant") {
      console.log("AI回复:", last.content);
      clearInterval(timer);
    }
  }, 1500);
}
run();
